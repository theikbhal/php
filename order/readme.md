# ⚡ FocusFlow Single-File PHP Engine

A hyper-focused, gamified task list designed specifically to counteract ADHD decision fatigue and task paralysis.

## 🚀 Features
- **Hard Focus Cap**: Limits active priorities to 5–10 items maximum.
- **Micro-Dopamine Rewards**: Built-in celebration triggers via Canvas Confetti.
- **Zero-Config DB**: Uses native SQLite file auto-generated on first run.
- **Multi-Format Export**: Export state instantly to `.json`, `.md`, or `.txt`.
- **Bulk Line Import**: Dump raw text lists instantly into the unorganized backlog.
